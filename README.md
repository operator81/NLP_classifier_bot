Предобработка проводится по следующим этапам: приведение к нижнему регистру, удаление спецсимволов и чисел, удаление пробелов и очистка от коротких сообщений (<10 символов).


При этом по оценке самого датасета после проведения разметки тональности на категории негатив (1-2 балла), нейтрально (оценка - 3) и позитив (4-5 баллов) получим следующее распределение отзывов:
- позитивные: 318151
- негативные: 165646
- нейтральные: 16095
- 
Проводится балансировка лкассов (из каждого класса берем одинаковое количество отзывов по наименьшему классу - 16095) и данные перемешиваются.

Проводится нормализация рейтинга к шкале 1-5.  

Далее векторизация TF-IDF с 3000 признаков с удалением редких слов (min_df=2)
